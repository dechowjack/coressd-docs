## Notes

- `OSU/` holds raw MODIS MOD10A1F Collection 6.1 tiles downloaded from NSIDC.
- `mwrzesie/` contains open-loop LIS runs produced by collaborators; these serve as the model prior.
- `Blender/Inputs/` is the staging area for pre-processed inputs ready for the Blender algorithm.
- `Blender/Runs/WYxxxx/temp_nc/` holds temporary per-slice outputs and **can be deleted** after the continental merge step.
- `installs/` contains Julia and conda environments shared across the project.
- `csd_dev/` must be the working directory when running the MATLAB preprocessing pipeline (see [Intermediary Processing](intermediary_processing.md)).
