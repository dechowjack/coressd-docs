## `csd_dev` MATLAB Functions

The `csd_dev` repository contains a `/functions/` subdirectory with preprocessing and helper scripts. The top-level functions are:

| Function | Description |
|---|---|
| `csd_main.m` | Main orchestration script — see [Intermediary Processing](intermediary_processing.md) |
| `csd_gen_mean_scf_files.m` | Generates SCF-A GeoTIFFs |
| `csd_gen_psval_nc.m` | Generates precipitation scalar NetCDF files |
| `csd_smooth_mass_vars.m` | Applies Gaussian smoother to mass variables (Snowf, SWE) |
